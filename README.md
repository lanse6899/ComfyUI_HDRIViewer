# 🔵BB HDRI Viewer

A ComfyUI custom node plugin for interactive 360-degree panorama viewing and image transformation.

一个 ComfyUI 自定义节点插件，用于交互式查看360度全景图像和图像变换处理。

## 🔵BB HDRI查看器 / HDRI Viewer

- 360° 全景图像交互式查看 / Interactive 360° panorama viewing
- 三种投影模式：球形3D、等矩形2D、立方体视图 / Three projection modes: Sphere 3D, Equirectangular 2D, Cube View
- 鼠标拖拽旋转、滚轮调整视角 / Mouse drag to rotate, scroll to adjust FOV
- 自动旋转、全屏模式 / Auto-rotate, fullscreen mode

## 🔵BB镜像图像 / Image Mirror

- 水平镜像 / Horizontal mirror
- 垂直镜像 / Vertical mirror
- 双向镜像 / Both directions

## Installation / 安装

```bash
# Copy to ComfyUI/custom_nodes/
cd ComfyUI/custom_nodes
git clone https://github.com/your-repo/ComfyUI_BB_HDRI.git
```

Restart ComfyUI. The nodes appear under category: `🔵BB HDRI viewer`

重启 ComfyUI。节点将显示在分类: `🔵BB HDRI viewer`
